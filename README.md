## Dealing with Installation Errors

| Error | Solution | 
|---|---|
| ModuleError: No 'kivy' ~ | try to reinstall 3.7 python or modify into noticible path  | 
| sdl2 - ImportError: DLL load failed | pip install kivy.deps.sdl2 & pip install kivy.deps.glew  |
